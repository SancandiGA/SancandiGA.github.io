I Web Fonts in CSS

Le web fonts sono un modo per utilizzare font personalizzati sul tuo sito web senza doverli installare sul computer dell'utente.

Per utilizzare una web font, devi prima caricarla sul tuo sito web utilizzando il tag <link> o il tag <style>. Ad esempio:

<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Tangerine">

Quindi, puoi utilizzare la web font nel tuo CSS utilizzando la proprietà font-family:

body { font-family: 'Tangerine', serif; font-size: 48px; background-size: cover; background-image: url('https://i.pinimg.com/originals/6d/45/f3/6d45f38dad7c0a9345e83b7e1df3c803.jpg')" }

Utilizzo di Google Fonts

Google Fonts è un servizio gratuito che offre una vasta gamma di web fonts. Puoi utilizzare Google Fonts aggiungendo un link al tuo file CSS:

<link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Tangerine">

Oppure, puoi utilizzare la proprietà @import nel tuo file CSS:

@import url('https://fonts.googleapis.com/css?family=Tangerine');

Stili e pesi

Le web fonts possono avere diversi stili e pesi. Ad esempio, la font Tangerine ha uno stile normale e uno stile corsivo:

https://fonts.googleapis.com/css?family=Tangerine:normal,italic

Oppure, puoi utilizzare la proprietà font-weight per specificare il peso della font:

body { font-family: 'Tangerine', serif; font-weight: bold; }
